\# Hydro Debouchage Wallonie



Site web pour entreprise de debouchage en Belgique (Wallonie).



\## Installation



npm install

npm run dev



\## Features



\- Landing pages dynamiques par ville

\- Chat assistant avec IA (Gemini)

\- Formulaire de contact avec EmailJS

\- Design responsive mobile-first



\## Stack



\- React 18

\- TypeScript

\- Tailwind CSS (CDN)

\- Vite

\- Google Gemini API

\- EmailJS
